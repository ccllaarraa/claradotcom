Construire un site web comme on construit une maison
HTML / CSS / Digital publishing / writing / web space as architecture


‘Construire un site web comme construire une maison’ est une initiation au développement web - nous allons apprendre à bâtir un site web à la main, en utilisant le HTML et CSS comme des matériaux bruts de construction.
À travers des exercices et des discussions sur le web d’aujourd’hui, nous serons amené·e·s à nous poser des questions types
‘À quoi ça sert d’avoir un site web quand on a Instagram?’
‘Pourquoi tous les sites se ressemblent-ils?’
‘Que faire de cet espace infini de la page web?’.
Nous nous questionnerons sur l'interface à travers multiples expérimentations afin de publier une page en ligne.

Consigne ◌
→ En utilisant le texte Wikipédia suivant sur l'interface: https://fr.wikipedia.org/wiki/Interface_utilisateur vous devrez créer une page web en republiant cet article en entier.
Vous créerez plusieurs pages web, et chaque page web comprendra le texte et un lien vers la page suivante créée.
→ Expérimentez - vous pouvez jouer avec les tailles de typographies, cacher le texte, le faire apparaitre en différé, faire des compositions en couleur, vous focaliser sur un élément de la page web et en exacerber son utilité (par ex faire une imposante scrollbar)……… tout est possible.
En utilisant un contenu répétitif, pensez ces pages comme une navigation à travers des atmosphères différentes. Vous pourrez par la suite ajouter des images, si elles ont un lien avec le texte.
→ À la fin de nos cours ensemble, vous devrez au moins avoir créé une navigation à travers 5 pages.
